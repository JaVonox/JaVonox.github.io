BankBot is a python-based machine learning algorithm designed to categorise and respond to questions relating to finance. Using spacy as a framework, the software parses and categorises an english language input into one of seven topics - using stack overflow questions and their respective tags as training data. When tested on said training data with a 70/30 split, the software consistently reported a 60-70% accuracy rate, with its second guess having a 20% accuracy rate. The software is designed to be extendable - allowing the training data, categories, responses and algorithms to be modified with some small changes to the files and or scripts. For this reason, the codebase is designed to be simple and legible, allowing for easy modification.
