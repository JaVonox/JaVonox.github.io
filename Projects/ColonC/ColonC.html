:C (Henceforth referred to as Colon C, as it is pronounced) is my attempt at creating a simple interpreted language using C# - arguably one of the most complicated solo projects I've ever worked on. While the language is simple, it carries a considerable amount of potential, as demonstrated by the AI blackjack project written entirely in the language. Included with the project is a file listing the available syntax, demonstrating the main goal of the project - developing a simple and streamlined language.
